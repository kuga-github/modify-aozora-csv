# modify-aozora-csv
Remove copyrighted books from original book list of Aozora Bunko.

## Getting started
### Requirement
* Python 3.x
* requests
* beautifulsoup4
### Run
```bash
python aozora.py
```
### File configuration
* `aozora.py`: source code
* `list_person_all_extended_utf8.csv`: input dada
* `index.csv`: output dada
